# Getting Started

Welcome to ObersUI! This section will get you from zero to a running app with creamy smooth components.

## What you'll need

- **Flutter 3.41+** and **Dart 3.11+**
- An editor (VS Code, Android Studio, or your favorite IDE)
- About 5 minutes

## The path ahead

1. [**Installation**](installation.md) — Add ObersUI to your project
2. [**Quick Start**](quick-start.md) — Set up `OiApp` and render your first widget
3. [**Project Structure**](project-structure.md) — Understand how the library is organized

After that, head to [Core Concepts](../core-concepts/index.md) to learn about the architecture, or jump straight into the [Component Catalog](../components/index.md) if you prefer to explore by doing.
