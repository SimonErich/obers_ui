# Interactive examples

ObersUI ships with interactive examples that let you explore components, toggle props, and see widgets respond in real time.

At the moment, this page is a placeholder until the new showcase fully replaces the old Widgetbook setup.

- Use the **examples** in the repo (`example/`) to run full apps locally.
- Use the **component docs** to see usage patterns, props, and code snippets.

Once the showcase is wired up, this page should link to it and document how to run it locally.
