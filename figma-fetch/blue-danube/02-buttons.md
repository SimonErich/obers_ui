# TapTap Design System - Buttons
## Source: Figma nodes 1162:22658 (Primary), cover button grid

## Button Variants

### Primary Button
- **Background:** #15C5CE (Primary 600)
- **Text:** White, PingFang SC Medium 500, 16px/24px
- **Padding:** 8px all sides
- **Border radius:** 4px
- **Min width:** 90px

#### States
| State | Background | Text | Border |
|-------|-----------|------|--------|
| Default | #15C5CE | White | none |
| Hover | #47CFD6 (Primary 500) | White | none |
| Pressed | #00ABB6 (Primary 700) | White | none |
| Disabled | #B0EBEC (Primary 300) | White | none |

### Outline Button
- **Background:** White (#FFFFFF)
- **Text:** #4B4B4B (Neutral 600), Regular 400, 16px/24px
- **Padding:** 8px all sides
- **Border:** 1px solid #E1E1E1 (Neutral 300)
- **Border radius:** 4px
- **Min width:** 90px

#### States
| State | Background | Text | Border |
|-------|-----------|------|--------|
| Default | White | #4B4B4B | 1px #E1E1E1 |
| Hover | White | #47CFD6 | 1px #47CFD6 |
| Pressed | White | #00ABB6 | 1px #00ABB6 |
| Disabled | White | #CACACA | 1px #EEE |

### Ghost Button
- **Background:** Transparent
- **Text:** #4B4B4B (Neutral 600), Regular 400, 16px/24px
- **Padding:** 8px all sides
- **Border radius:** 4px
- **Min width:** 90px

#### States
| State | Background | Text |
|-------|-----------|------|
| Default | Transparent | #4B4B4B |
| Hover | #F5F5F5 (Neutral 100) | #4B4B4B |
| Pressed | #E1E1E1 (Neutral 300) | #4B4B4B |
| Disabled | Transparent | #CACACA |

### Link Button
- **Background:** none
- **Border:** none
- **Text:** Regular 400, 16px/24px

#### States
| State | Text Color |
|-------|-----------|
| Default | #15C5CE (Primary 600) |
| Hover | #47CFD6 (Primary 500) |
| Pressed | #00ABB6 (Primary 700) |
| Disabled | #7DDDE1 (Primary 400) |

### Danger/Destructive Button (Outline variant)
- **Background:** White
- **Text:** #F64C4C (Danger 600), Regular 400, 14px/22px
- **Border:** 1px solid #F64C4C
- **Border radius:** 4px
- **Min width:** 82px
- **Padding:** 8px horizontal, 7px vertical

### Small/Compact Buttons
- **Text:** 14px/22px (vs 16px/24px for default)
- **Min width:** 82px (vs 90px for default)
- **Padding:** 8px horizontal, 7px vertical
