# TapTap Design System - Typography
## Source: Figma cover page node 1162:22536

## Font Families
- **Primary:** PingFang SC
  - Regular (weight 400) — body text, captions
  - Medium (weight 500) — titles, buttons, strong text
- **Monospace/Numbers:** Roboto Bold (weight 700) — used for large display numbers
- **Note:** For Flutter, we'll use system default or a similar sans-serif font

## Type Scale

| Style | Size | Line Height | Weight | Letter Spacing |
|-------|------|-------------|--------|---------------|
| H1 | 30px | 38px | Regular 400 | 0 |
| H2 | 24px | 32px | Regular 400 | 0 |
| H3 | 20px | 28px | Regular 400 | 0 |
| Title | 18px | 26px | Medium 500 | 0 |
| Subtitle | 16px | 24px | Medium 500 | 0 |
| Body | 14px | 22px | Regular 400 | 0 |
| Caption | 12px | 18px | Regular 400 | 0 |

## Weight Variants
- **Regular 400** — Default for most text
- **Medium 500** — Emphasis, titles, buttons, strong labels

## Usage Patterns
- **Button text (default):** Subtitle - Medium 500, 16px/24px
- **Button text (small):** Body - Regular 400, 14px/22px
- **Dialog title:** Title - Medium 500, 18px/26px
- **Dialog body:** Body - Regular 400, 14px/22px
- **Toast text:** Body - Regular 400, 14px/22px
- **Badge text:** Caption - Regular 400, 12px/18px
- **Slider marks:** Caption - Regular 400, 12px/18px
- **Tooltip text:** Body - Regular 400, 14px/22px
